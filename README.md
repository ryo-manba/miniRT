# miniRT

## Description
A Raytracer written in C and MiniLibX.

## Requirement

- gcc or Clang
- GNU Make

## Usage
```
git clone https://github.com/ryo-manba/miniRT
cd miniRT && make
```
An executable named miniRT will be created.
```
./miniRT [RTFILE_PATH]
```
